NSA:Codenames

Weclome to NSA:Codenames, a curated dataset of codenames used by the NSA, as reported by the media.

Learn more

This site only lists codenames that have been covered by mainstream media outlets. It is not a leak site and does not perform any primary research or journalism.

If you're not sure where to start, then we suggest that you peruse the alphabetical index of codenames or search for a particular codename.

If you are aware of a codename that is not contained in this database, please send us a link to a reputable news source using our issue tracker.
